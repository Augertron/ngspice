CUSPICE
ngspice for CUDA

compiled with MS Visual Studio 2015 Community and CUDA toolkit 8.0 for Windows

derived from branch CUSPICE
XSPICE and CIDER are not included
only 64 biut compilation is possible because runtime library cusparse64_80.dll is only 64 bit.

To compile, install VS 2015 and CUDA toolkit 8.0 from https://developer.nvidia.com/cuda-downloads.

To distribute the executable, add CUDA distributable run-time libraries cudart64_80.dll and cusparse64_80.dll.
